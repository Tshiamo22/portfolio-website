# Personal Portfolio Website
https://tshiamo22.github.io/portfolio-website/

This is my first front-end project built using HTML and CSS.

The website showcases my background, skills, experience, and contact information.
It focuses on clean layout, responsiveness, and modern CSS practices.

## Features
- Responsive design
- Flexbox-based layouts
- Clean and simple UI
- Multiple sections (About, Skills, Experience, Contact)

## Technologies Used
- HTML5
- CSS3

## What I Learned
- Structuring a multi-section webpage
- Using Flexbox for horizontal and vertical alignment
- Creating responsive layouts with media queries
- Improving UI consistency and spacing

## Future Improvements
- Add JavaScript for interactivity
- Improve accessibility
- Enhance animations and transitions
